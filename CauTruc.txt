====================================================
CẤU TRÚC HỆ THỐNG - Color Wheel VR/AR/PC Game
Three.js r170 + WebXR + Meta Quest 3 + PC Browser
====================================================

📁 ROOT
├── index.html          Entry point. Load main.js, chứa canvas WebXR.
│                       CSS: styled VR/AR/PC buttons, crosshair ẩn/hiện theo
│                       pointer lock, instructions card đổi nội dung theo mode.
│                       Nút #PCButton: fixed bottom center, click → activatePCMode().
├── main.js             Khởi tạo SceneManager, chọn mode VR/AR/PC, vòng lặp animation.
│                       Gọi GunMode.updateButtonInput() và GunMode.update(delta) mỗi frame
│                       khi ở VR/AR. Gọi pcMode.update(delta) và gunMode.update(delta)
│                       khi ở PC mode. grabSystem.update(delta) chạy ở mọi mode.
│                       activatePCMode(): gọi vrMode.enter() để build room, tạo PCMode,
│                       build ColorCircle tại vrMode.wallAnchor, spawn spheres.
│                       Render loop guard: camera auto-orbit chỉ chạy khi
│                       !activeMode && !pcMode.
├── vite.config.js      Cấu hình Vite: HTTPS (basicSsl), host LAN, build esnext.
├── package.json        Dependencies: three, vite, plugin-basic-ssl. Scripts: dev/build/preview.
├── vercel.json         Deploy config: buildCommand, outputDirectory, rewrites.
└── .gitignore          Bỏ qua: node_modules, dist, .env, .vercel.

📁 core/
├── SceneManager.js     Tạo scene, camera, renderer WebXR. Quản lý switch session
│                       immersive-vr / immersive-ar. referenceSpace: local-floor.
│                       Thêm: isPCMode flag + enablePCMode() tắt renderer.xr.enabled.
│
├── SphereGenerator.js  Array 12 màu HSL preset (bước 30°). Spawn sphere random
│                       position mỗi level.
│                       userData.color lưu hex HSL.
│                       userData.colorIndex lưu index (0–11) tương ứng COLOR_PRESETS.
│                       Cả hai field bắt buộc để checkDrop() hoạt động đúng.
│
├── LevelManager.js     Mỗi level = 1 ColorCircle (Group, 12 slot trên vòng tròn).
│                       3 slot primary (0°/120°/240°) tô màu sẵn, không tương tác.
│                       9 slot còn lại rỗng, userData.expectedColorIndex.
│                       Win condition: tất cả slot active filled đúng → next level.
│                       buildColorCircle() nhận THREE.Vector3 (không nhận Object3D).
│                       checkDrop(): so sánh sphere.userData.colorIndex với
│                         slot.expectedColorIndex — dùng index số, không dùng hex string.
│                       fillSlot(): set slot.filled=true, sphere.visible=false,
│                         thêm fillDisc vào colorCircle, kích hoạt animation + win check.
│
├── GrabSystem.js       Shared grab/drop/physics logic cho cả VR và PC mode.
│                       Thay thế Interaction.js cũ — dùng chung 1 class cho mọi mode.
│
│                       grabSphere(handIndex, handObject):
│                         Tìm sphere gần nhất trong activeSpheres (GRAB_RADIUS=0.25m).
│                         Skip nếu isLocked hoặc isGrabbed. Attach vào handObject.
│                         Right hand không grab khi gunMode.isActive.
│
│                       releaseSphere(handIndex, handObject, otherHandObject):
│                         Tính velocity qua VelocityTracker.
│                         Left thả gần right + gun active → loadAmmo(color, colorIndex).
│                         Truyền cả hex và colorIndex vào GunMode.loadAmmo().
│                         Drop bình thường → checkDrop → _onCorrectDrop / _onWrongDrop.
│
│                       handOffToPhysics(sphere, velocity):
│                         Public method. GunMode gọi khi sphere bắn sai màu để
│                         chuyển giao sphere về GrabSystem physics pipeline.
│                         Sphere tiếp tục được simulate, checkDrop, và grabbable lại.
│
│                       _onCorrectDrop(sphere, slot):
│                         playCorrect() + fillSlot() + xóa khỏi activeSpheres.
│
│                       _onWrongDrop(sphere):
│                         playFail() + _rejectSphere() (animation đẩy ra).
│                         Sphere vẫn trong activeSpheres, grabbable lại bình thường.
│
│                       update(delta):
│                         Track velocity sphere đang cầm.
│                         Simulate physicsBodies: checkDrop mỗi frame,
│                         xóa sphere rơi y < -2 khỏi scene + activeSpheres.
│
├── PhysicsBody.js      Inertia system cho sphere sau khi thả hoặc bắn.
│                       PhysicsBody: apply velocity + gravity (-4 m/s²) + damping
│                       (0.92/frame) + floor clamp (y=0.05) + bounce nhỏ.
│                       Dừng khi speed < 0.02. Tự xóa khỏi physicsBodies[].
│                       VelocityTracker: rolling buffer 4 frame world position,
│                       tính velocity trung bình để tránh spike khi tay giật cục.
│
├── VRInput.js          Wrapper XR controller events → gọi GrabSystem.grabSphere/
│                       releaseSphere với controller Object3D tương ứng.
│                       Tách biệt input layer khỏi logic grab/drop.
│
├── AudioManager.js     1 AudioListener gắn camera. 3 PositionalAudio:
│                         correct.mp3 → phát tại vị trí slot.
│                         fail.mp3    → phát tại vị trí sphere.
│                         ambient.mp3 → loop, volume thấp, gắn scene center.
│
└── GunMode.js          Chế độ bắn sphere bằng súng. Nhận SceneManager,
                        SphereGenerator, LevelManager, GrabSystem.

                        VR/AR toggle: nút A (right controller, index 4).
                        PC toggle: gọi toggle({ attachTo: rightHand }).
                        Bật: hiện model súng DEpistol.glb.
                          VR: gắn vào right grip.
                          PC: gắn vào rightHand Object3D.
                        Tắt: ẩn súng, xóa đạn đang nạp.

                        Nạp đạn (loadAmmo(hexColor, colorIndex)):
                          Nhận hexColor + colorIndex từ GrabSystem khi left thả
                          sphere gần right hand. Lưu cả _ammoColor và _ammoColorIndex.
                          Hiện ammoIndicator (PlaneGeometry 0.06m) đổi màu theo đạn.

                        Bắn (fire(originObject)):
                          Tạo sphere tại world position của originObject.
                          sphere.userData.color = hex, userData.colorIndex = index.
                          BULLET_SPEED = 16 m/s theo hướng ngắm originObject.
                          VR: originObject = right controller.
                          PC: originObject = camera.

                        Update mỗi frame:
                          Simulate PhysicsBody các sphere đang bay.
                          Check drop vào slot qua LevelManager.checkDrop():
                            Đúng màu → fillSlot + scene.remove(sphere) +
                                        xóa activeSpheres + haptic 0.8/200ms.
                            Sai màu  → tính bounceVel (reflect + 0.4) →
                                        grabSystem.handOffToPhysics(sphere, bounceVel).
                                        GrabSystem tiếp quản: simulate tiếp, grabbable lại.
                          Xóa sphere nếu y < -2 (rơi khỏi scene).

                        Assets:
                          /assets/DEpistol.glb → model súng (scale 0.05).
                          ammoIndicator: PlaneGeometry, DoubleSide, opacity 0.85,
                          gắn right controller tại (0.08, 0, -0.05), xoay π/4.

📁 modes/
├── VRMode.js           Virtual room: BoxGeometry 6 mặt, MeshPhongMaterial.
│                       Accent strip lights + star field particles.
│                       wallAnchor: Vector3(0, 2, -3.95) để đặt ColorCircle.
│                       Left joystick: locomotion qua VRInput / GrabSystem.updateLocomotion.
│                       Dùng XRControllerModelFactory render model tay cầm.
│                       enter()/exit() public — PC mode gọi enter() để dùng room.
│
├── ARMode.js           Session: immersive-ar + hit-test + local-floor.
│                       XRHitTestSource từ viewer space → detect mặt phẳng sàn.
│                       Tap đặt ColorCircle anchor cố định trên sàn thực.
│                       Passthrough, không tạo virtual room.
│                       Sphere spawn quanh vị trí anchor.
│
└── PCMode.js           Chế độ chơi trên PC browser (không cần WebXR).
                        Tự động kích hoạt nếu immersive-vr không support,
                        hoặc thủ công qua nút #PCButton.

                        Camera/Look:
                          Pointer Lock API (click canvas để lock, ESC để thoát).
                          mousemove → cập nhật _yaw / _pitch (giới hạn pitch ±π/2.2).
                          Camera rotation chỉ apply khi _isLocked = true.

                        Movement:
                          WASD / Arrow keys → translate theo hướng _yaw (horizontal).
                          Space → lên Y, C → xuống Y. MOVE_SPEED = 3.0 m/s.

                        Virtual hands:
                          leftHand, rightHand: Object3D gắn vào camera.
                          Offset cố định: L(-0.18, -0.18, -0.5) R(0.18, -0.18, -0.5).
                          Visual: BoxGeometry nhỏ, màu xanh (left) / cam (right).

                        Input mapping:
                          Click trái  → leftHand grab (GrabSystem) / fire() khi gun active.
                          Click phải  → rightHand grab (GrabSystem).
                          Q           → toggle GunMode({ attachTo: rightHand }).
                          Left thả gần rightHand + gun active → GrabSystem.releaseSphere
                                        tự detect và gọi loadAmmo(color, colorIndex).

                        Crosshair:
                          SVG crosshair inject vào DOM (#pc-crosshair).
                          Hiện khi pointer locked, ẩn khi unlock.

                        dispose(): exitPointerLock, xóa crosshair, detach hands.

====================================================
COLOR SYSTEM
====================================================
Bậc 1 - Primary (slot có màu sẵn):
  index 0  = Red     #FF0000  (0°)
  index 4  = Green   #00FF00  (120°)
  index 8  = Blue    #0000FF  (240°)

Bậc 2 - Secondary (slot trống):
  index 2  = Yellow    #FFFF00  (60°)
  index 6  = Cyan      #00FFFF  (180°)
  index 10 = Magenta   #FF00FF  (300°)

Bậc 3 - Tertiary (slot trống):
  index 1  = Orange        (30°)   index 3  = Chartreuse   (90°)
  index 5  = Spring Green  (150°)  index 7  = Azure        (210°)
  index 9  = Violet        (270°)  index 11 = Rose         (330°)

QUAN TRỌNG: checkDrop() so sánh sphere.userData.colorIndex (số nguyên 0–11)
với slot.expectedColorIndex. Mọi nơi tạo sphere (SphereGenerator, GunMode.fire())
phải set userData.colorIndex đúng. Không dùng hex string để match màu.

====================================================
GUN MODE - LUỒNG HOẠT ĐỘNG
====================================================
VR/AR:
1. Nhấn nút A (right controller) → toggle GunMode.
2. Left tay cầm sphere → thả vào right controller → GrabSystem detect →
   loadAmmo(hexColor, colorIndex).
3. ammoIndicator đổi màu hiển thị đạn đang nạp.
4. Nhấn trigger right → fire(rightController): sphere bay theo hướng right controller.
5. Sphere trúng slot đúng màu → fillSlot + scene.remove + haptic mạnh.
   Sphere trúng slot sai màu → handOffToPhysics → GrabSystem tiếp quản →
                                _onWrongDrop → rejectAnim → grabbable lại.
   Sphere rơi xuống y < -2   → tự xóa khỏi scene.

PC:
1. Nhấn Q → toggle GunMode({ attachTo: rightHand }).
2. Click trái cầm sphere → thả gần rightHand → loadAmmo(color, colorIndex).
3. ammoIndicator đổi màu.
4. Click trái → fire(camera): sphere bay theo hướng nhìn camera.
5. Kết quả trúng/trượt giống VR, không có haptic.

====================================================
PC MODE - ĐIỀU KHIỂN
====================================================
Click canvas       → bắt Pointer Lock, hiện crosshair
ESC                → thoát Pointer Lock
Chuột              → nhìn xung quanh (yaw/pitch)
W / A / S / D      → di chuyển tiến/trái/lùi/phải
Space              → lên cao
C                  → xuống thấp
Click trái         → tay trái grab sphere / bắn (gun mode)
Click phải         → tay phải grab sphere
Q                  → toggle Gun Mode
